II. Hướng dẫn chạy source code
- Phần mềm sử dụng: Spring Tool Suite 4, XAMPP.
- Kết nối Database:
 	B1: Mở phần mềm XAMPP, nhấn chọn "Start" ở Apache và MySQL.
	B2: Chọn "Admin" ở MySQL.
	B3: Tại trang phpMyAdmin, tạo một database mới tên là "demo_web"
	Lưu ý: Đặt đúng tên database để có thể kết nối thành công.

- Chạy chương trình:
	B1: Sau khi mở project trên Spring Tool Suite, chuột phải vào project.
	B2: Chọn "Run as", sau đó nhấn chọn "Spring Boot App" và đợi chương trình chạy.
	B3: Sau khi hoàn tất chạy chương trình, mở trình duyệt và truy cập "localhost:8081".
	B4: Sau khi truy cập "localhost:8081", màn hình sẽ hiển thị trang chủ của trang web.
